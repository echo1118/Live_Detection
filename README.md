#### 一、简介
- 基于dlib；
- 基于OpenCV；
- 依赖于shape_predictor_68_face_landmarks.dat（用于人脸68个关键点检测的dat模型库）
- 实现实时活体检测功能，包含眨眼检测、张嘴检测、摇头检测、点头检测；

#### 二、功能实现

- 眨眼检测：基于计算眼睛纵横比；
- 张嘴检测：基于计算嘴唇纵横比；
- 摇头检测：基于计算鼻子到面部轮廓的距离对比；
- 点头检测：基于计算左眉毛与左右脸轮廓的欧氏距离(两边之和是否小于或等于第三边+阈值)；
